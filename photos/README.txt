HOW TO ADD YOUR ACHIEVEMENT PHOTOS
==================================

Drop your images into THIS folder (project/photos/) using these exact names.
As soon as a file exists, it replaces the "REPLACE WITH YOUR IMAGE" placeholder
on that award card. If a file is missing, the placeholder shows instead.

  pentathon-2025.jpg      -> Pentathon 2025 card
  byteme-ctf-2026.jpg     -> ByteMe CTF 2026 card
  commit-1-hackathon.jpg  -> Commit 1.0 Hackathon card
  tenet-ctf-2025.jpg      -> TENET CTF 2025 card

Tips
----
- Each photo fills HALF the screen (a tall portrait-ish area). Landscape photos
  also work — they are cropped to fill (object-fit: cover), centered.
- Recommended size: at least 1000 x 1400 px for sharpness on large screens.
- .jpg, .png and .webp all work. If you use a different extension, update the
  matching <img src="..."> in Portfolio.dc.html (search for "./photos/").
